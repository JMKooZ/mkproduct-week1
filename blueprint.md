
# 로또 번호 생성기

## 개요

이 애플리케이션은 사용자에게 5개의 무작위 로또 번호 조합을 생성하여 제공합니다. 버튼을 클릭하면 중복되지 않는 6개의 숫자로 이루어진 5개 세트가 화면에 표시됩니다.

## 디자인 및 기능

*   **레이아웃:** 중앙 정렬된 단일 컬럼 레이아웃.
*   **스타일:** 현대적이고 깔끔한 디자인을 적용합니다. 결과가 표시되는 영역은 시각적으로 구분되도록 스타일링합니다.
*   **기능:**
    *   "번호 생성" 버튼을 클릭하면 이벤트가 트리거됩니다.
    *   1부터 45까지의 숫자 중에서 무작위로 6개의 고유한 숫자를 선택합니다.
    *   위 과정을 5번 반복하여 총 5개의 번호 세트를 생성합니다.
    *   생성된 번호는 정렬하여 화면에 명확하게 표시합니다.

## 현재 계획

1.  **HTML (`index.html`):** 페이지의 기본 구조를 설정합니다. 제목, 번호가 표시될 컨테이너, 그리고 번호 생성을 위한 버튼을 포함합니다.
2.  **CSS (`style.css`):** 애플리케이션의 디자인을 담당합니다. 버튼, 컨테이너 및 전체적인 레이아웃에 스타일을 적용하여 시각적으로 매력적이고 사용자 친화적으로 만듭니다.
3.  **JavaScript (`main.js`):** 로또 번호 생성 로직을 구현합니다. 버튼 클릭 이벤트를 감지하고, 무작위 번호를 생성하여 화면에 동적으로 렌더링합니다.
